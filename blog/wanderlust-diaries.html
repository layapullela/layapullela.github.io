Graph Visualizer

A tool I developed to visualize complex graph structures commonly used in bioinformatics and computational genomics. This interactive web-based application allows researchers to:

  • Upload adjacency matrices or edge lists from genomic network data
  • Visualize protein-protein interaction networks with customizable node attributes
  • Apply various graph layout algorithms optimized for biological networks
  • Perform common graph analytics such as centrality measures and community detection
  • Export high-resolution images for publications

Built with D3.js and React, this tool has been particularly useful for visualizing gene regulatory networks and understanding complex relationships in genomic data. The project incorporates several optimization techniques to handle large-scale biological networks with thousands of nodes.

Technologies used: JavaScript, D3.js, React, Python (backend), NetworkX
